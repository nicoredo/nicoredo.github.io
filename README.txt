# MedReg miniweb (subruta /medreg)

Esta carpeta está lista para subirse al repositorio de tu sitio `medex.ar`.
Ubícala en la raíz del repo como `medreg/` y activa GitHub Pages para `main`.
Tu demo quedará en: `https://medex.ar/medreg/` (si el dominio custom ya apunta).

Archivos clave:
- `index.html`: página demo (UI del sidebar).
- `chrome-shim.js`: evita errores si no hay extensión (modo demo).
- `medreg_local_demo.js`: lógica del demo.
- `terminologia_medica.json`, `reglas_cruce.json`: datasets para el demo.
- `icon*.png`: íconos.
